CSE 400 Thesis

Context-Aware Zero-Shot Anomaly Detection in Surveillance Using Contrastive and Predictive Spatiotemporal Modeling

Supervisors: Prof. Dr. Md. Ashraful Alam, Md Tanzim Reza

Deep Learning Overfitting Optimization

Designed a Zero-Shot, context-aware video anomaly detection framework that detects unseen abnormal events without requiring anomaly samples during training, leveraging spatiotemporal prediction and semantic context.

Tech Stack: Python, PyTorch, Hugging Face Transformers, TimeSformer, CLIP, DPC-RNN, CNN, Contrastive & Zero-Shot Learning, InfoNCE/CPC losses, OpenCV, UCF-Crime dataset.

Proposed a dual-stream architecture combining TimeSformer based spatiotemporal transformers, DPC-based predictive modeling, and CLIP visionâ€“language alignment to enable pure zero-shot anomaly detection.

The model outperformed multiple state-of-the-art zero-shot and vision-only baselines while reducing false alarms via scene-aware context conditioning.

CSE 424 (Pattern Recognition) Project

Blue-Light Blocking Glasses Using Machine Learning

Instructors: Annajiat Alim Rasel, Sadiul Arefin Rafi

Computer Vision Biomedical Imaging Representation Learning Multi-Modal Fusion

Tech Stack: Python, Scikit-learn, Pandas, NumPy, Matplotlib, OpenRefine, SVM, KNN, Linear Regression, Spectrophotometric Data Analysis.

Analyzed 50 commercial blue-light blocking lenses under 5 light sources, measuring transmission across 380-780nm visible spectrum. The K-NN model achieved the highest accuracy, outperforming SVM and Linear Regression.

Research demonstrating orange-tinted glasses exhibited highest transmission specificity for regulating circadian sleep-wake rhythms.
