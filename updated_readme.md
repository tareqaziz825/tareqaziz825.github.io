## Local Development

To run this website locally:

1. Clone the repository:
```bash
git clone https://github.com/tareqaziz825/tareqaziz825.github.io.git
cd tareqaziz825.github.io
```

2. Open `01index.html` in your browser:
```bash
# On macOS
open 01index.html

# On Linux
xdg-open 01index.html

# On Windows
start 01index.html
```

Or use a local server:
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

Then visit `http://localhost:8000/` in your browser.
