CSE 424 (Pattern Recognition) Project

Blue-Light Blocking Glasses Using Machine Learning

Instructors: Annajiat Alim Rasel, Sadiul Arefin Rafi

Computer Vision Biomedical Imaging Representation Learning Multi-Modal Fusion

Tech Stack: Python, Scikit-learn, Pandas, NumPy, Matplotlib, OpenRefine, SVM, KNN, Linear Regression, Spectrophotometric Data Analysis.

Analyzed 50 commercial blue-light blocking lenses under 5 light sources, measuring transmission across 380-780nm visible spectrum. The K-NN model achieved the highest accuracy, outperforming SVM and Linear Regression.

Research demonstrating orange-tinted glasses exhibited highest transmission specificity for regulating circadian sleep-wake rhythms.

CSE 427 (Machine Learning) Project

Loan Eligibility Prediction Using Machine Learning Models with SMOTE for Class Imbalance

Instructor: Prof. Dr. Chowdhury Mofizur Rahman

Classification SMOTE Random Forest XGBoost

Tech Stack: Python, Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn, Imbalanced-learn, XGBoost, LightGBM.

Developed a machine learning pipeline for loan eligibility prediction addressing class imbalance using SMOTE (Synthetic Minority Over-sampling Technique).

Evaluated multiple classifiers including Logistic Regression, Random Forest, XGBoost, and LightGBM with hyperparameter tuning and cross-validation to optimize performance metrics.

Achieved improved recall for minority class while maintaining high overall accuracy, providing a robust solution for financial institution loan approval systems.
